
# 🚗 Smart Route Optimizer

AI–powered web application for traffic prediction and route optimization. It delivers accurate travel time estimation (ETA) and intelligently suggests routes by avoiding congestion while strictly following real-world road networks.

---

## ✨ Features

* 🚦 Traffic prediction (Low, Medium, High)
* ⏱️ Travel time estimation (ETA)
* 🧭 Smart route optimization (avoid congested roads)
* 🗺️ Interactive and user-friendly map interface
* 📊 Uses historical and simulated data

---

## 🛠️ Tech Stack

* **Frontend:** Vite + TypeScript
* **Styling:** Tailwind CSS
* **Testing:** Vitest, Playwright
* **ML Approach:** Regression-based prediction

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Habtamu91/smart-route-optimizer.git
cd smart-route-optimizer
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
